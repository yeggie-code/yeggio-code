### 🚀 About Me

IT graduate passionate about transforming data into actionable insights. I enjoy building data-driven solutions with Power BI, SQL, Python, and AI while continuously learning new technologies. I’m driven by solving real-world problems, creating meaningful dashboards, and using technology to improve business decisions.

🔭 &nbsp;I'm currently working on **Building Power BI dashboards, strengthening my data analytics portfolio, and developing real-world data projects.**  
🌱 &nbsp;I'm currently learning **Advanced SQL, Data Engineering, Machine Learning, and cloud technologies for modern analytics.**  
👯 &nbsp;I'm looking to collaborate on **pen-source data analytics projects, AI applications, business intelligence dashboards, and data visualization initiatives.**
